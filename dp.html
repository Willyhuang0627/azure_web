DP 系列認證介紹

DP (Data Platform) 系列認證是微軟針對資料工程、資料分析與資料科學相關能力設計的 Azure 認證。

適合對象

  • 資料分析師
  • 資料工程師
  • 資料科學家
  • 想學習 Azure 資料平台的人
  • 從事資料處理或資料分析相關工作者

常見 DP 認證

  • DP-900：Azure Data Fundamentals
  • DP-100：Azure Data Scientist Associate
  • DP-203：Azure Data Engineer Associate

準備方向

  • 學習資料庫基礎 (SQL)
  • 了解資料處理流程
  • 熟悉 Azure 資料服務
  • 學習 Python 或資料分析工具
  • 練習官方模擬試題
